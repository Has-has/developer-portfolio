## Getting Started

Clone the repository:

```bash
git clone https://github.com/Has-has/portfolio.git
```

Navigate into the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
